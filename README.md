# NewsFlush App

## Get Started

NewsFlush is a React Vite app that pulls news articles from the News API.

### Installation

```bash
npm install
```

### Running the App

```bash
npm run dev
```
This will start the frontend app.

### Navigation (News)

All news articles are found in the **News** section in the navbar. It contains articles on:
- Technology
- Science
- Entertainment
- Sports
- Business
- and more.
